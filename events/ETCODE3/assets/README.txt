Place your base certificate images for the ETCODE3 event in this folder.

Expected filenames used by the current scripts:
- participant.png  (base image for participant certificates)
- organizer.png    (base image for organizer certificates)

You can add other role-specific images (e.g. contributor.png) and wire them
by updating events/ETCODE3/config.json and the generate/send scripts if needed.
